# JavaScript之面向对象编程

JavaScript语言上手非常容易，本身也不需要什么庞大复杂的开发环境，这样的特点成就了互联网上的第一大语言。

但是，一旦想在JavaScript中使用面向对象编程，就会发现其中的一些实现非常绕，理解起来也很烧脑。本系列侧重于呈现JavaScript中面向对象编程的实现，希望能够从概念上对JavaScript面向对象编程有正确的理解，从而达到消除各种误解的目的。
